看《象与骑象人》的领悟

如果理智的我是骑象人，而本能欲望是大象。
很多时候事与愿违，是因为没有驾驭大象的能力。

如果说这个是世界，没有事实，只有阐释。
那么我的一切痛苦来源，都是来源于自己对世界的解读。

这本书里面讲，人的幸福度，主要来源于：基本的物质保障，爱人的陪伴，稳定的人际关系
改变思维的方式：
1.冥想：冥想可以让人静下来，能减轻焦虑
2.认知疗法：就是看心理医生。很多人心理出了问题，就是因为他对自我的认知出现了偏差，不能用正常的思维去看问题，都是沮丧。
比如，我这个人很糟糕，我的世界一片黑暗，我的未来毫无希望。
比如在公众场合讲错一句话，就觉得自己一无是处，连话都讲不好，一直责怪自己，而沮丧忧郁时候思考，会扭曲事实，进而产生消极情绪，而消极的情绪又让扭曲的思考更为严重，形成恶性循环。
打破这种循环，就是记录想法，然后把扭曲之处找出来，用出正确的思考方式代替。

当你比较混乱，不知道怎么办才好的时候，可以去图书馆看书，可以去逛街，可以去运动，让自己静下来，而不是躺在床上胡思乱想。

这本书，对我来说，最有用的就是对负面偏好的解读，以及解决方法。
我是一个有很强负面偏好的人，还有很强的内在自己我攻击。
总认为自己，一无是处，一无所有，人生无望。
一句话的不得当，我都会脸红，羞愧。
总是怪自己，认为都是自己的错。

这种想法，就是书中讲的沮丧忧郁，脑中自动化思考，充斥种种毫无建设性的灰色念头，尤其是事情出错的时候。
我也是今年才发现自己，几乎所有苦痛情绪的来源，都是这些认知失调。
治疗这种问题，需要改变自己的思维方式。

我自己找到的方式
第一步：静下来
不管你如何恐惧，焦虑，不安，都让自己静下来。
这一步特别难，如果我不知道自己在恐惧什么，那就冥想。
如果能感受到恐惧的东西，那就写文字，把问题一条一条的写下来。
因为只有当你真正的静下来，才能理性的去看待事实，基于现实思考如何解决问题。
（大概在2024年的时候，我真的觉得自己人生毫无希望，想着跳湖结束，因为一个契机或者是别的原因，我又熬了一段时间，今年，在回过头看的时候，有一个很深刻的体会：这个世界没有绝路，更加没有思路，你永远都有选择的权力，你会绝望，是被困在当下那个情绪里了，等熬过那段时间就好了）
因为这件事情，我意识到，冷静下来，静下来，是解决所有问题的第一步。
第二步：改变自己的思维
如果你还是和以前一样思考，那么，此刻的你，只是重复的经历着，以前痛苦的循环，并不会有任何成长。
所以真正的改变，应该是改变自己的思维。
最好的办法，就是找到你认知失调具体根源，并且用正确的代替。
找医生，看书，接触观察新的正常长人，这些都可以。
第三步：行动
这一步比较简单，不要怕，不要思考，按照计划去做。
所有的改变，正确与否，在你的行动中，自然会有答案。
这三个步骤，反复循环反思。
我坚定的相信，思维改变自然而然，水到渠成。

如何快速搭建网站

准备工具：（node.js git hexo GitHub账号）

1.下载安装node
  • node.js下载官方地址：https://nodejs.org/en/download/
  • 安装之后打开cmd输入代码检查是否安装成功：node -v
2.下载安装git

git下载官方地址： https://git-scm.com/download/win
安装之后打开cmd
输入代码检查是否安装成功：git -v
vscode如何预览markdown

3.用管理员打开cmd

输入代码下载hexo：npm install hexo-cli -g

4.GitHub里面创建一个仓库
5.关联git与GitHub（配置ssh key）
  • 桌面打开git bash here
  • 运行:ssh -keygen -t rsa -c “你的邮箱” 运行
  • 进入文件夹的这个位置：C:\Users\2017.4.29.ssh 复制 id_rsa.pub的内容
  • 打开GitHub=》settings=》SSH and GPG keys =》New key
    标题任意，把内容复制到key里面

桌面打开git bash here输入：ssh -T git@github.com 验证是否添加成功

6.本地部署
  • 选择一个合适地址，创建一个文件夹，用来放置博客文件
  • 右击打开文件夹的git bash here
  • 输入：
    hexo init 初始化
    npx hexo install 安装
    npx hexo g 生成
    npx hexo s 本地部署
打开跳出来的链接，就能看到本地部署成功了

ctrl+c停止本地服务器

7.线上部署

  • 打开博客文件夹中的config文件

  • 把这段文字复制替换
    type:git
    repository:
    branch:main
    （需要注意文字缩进格式）

  • 打开GitHub复制code的https链接

  • 粘贴在repository后面,保存退出

  • 打开git bash here
    npx hexo g 生成
    npx hexo d 上传
    第一次使用需要配置
    git config –global user.email “你邮箱”

    git config –global user.name “你的名字”

  • 重新上传 npx hexo d

打开在GitHub上面粘贴的code地址，就能看线上项目了

真正的选择权，是拥有离开的权力

以前我理解的主动权是解决问题
遇见问题，你主动找对方沟通，解决
昨天一个事，发现自己看问题的角度不一样了

这两天刚搬家，合租，特别不隔音，隔壁刷视频的声音都能听清楚的那种。

刚来上海，前几天都有事，每天几乎只睡差不多两三个小时

今天想着早睡一点，九点就躺床上了，但隔壁一直聊天，很兴奋那种

我很无语，也很无奈

默默的去拿降噪耳机，带上

以前这种情况，我是愤怒的，生气的，想着和对方沟通的，想的都是如何让他闭嘴。

整个人的状态会是愤怒的
现在静静的思考后，其实所谓的主动权，他包含了，你解决问题的能力。

但是，他更应该是你对自己人生权的把控

对于有噪音的这个问题

解决的思路不在于和对方怎么沟通

而是，你要足够的努力，让自己尽早的离开这个地方，离开那些让你不开心的人，地方，事物，而不是在那些看似小事，实则无解的事物上，无止尽的消耗自己的精力。

人最痛苦的莫过于，你不喜欢，但离不开。
改变的第一步，静下来，接受最烂的自己
